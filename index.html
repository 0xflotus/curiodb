CurioDB

CurioDB is a distributed and persistent NoSQL database, featuring:

  • Redis compatible API
  • Distributed transactions
  • Embedded Lua scripting
  • Virtual memory
  • Pluggable storage
  • TCP, WebSocket, HTTP interfaces
